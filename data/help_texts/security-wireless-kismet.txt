Usage: kismet [options]

Options:
  -c, --config=<file>         Path to configuration file
  -s, --server=<host>         Kismet server hostname/IP
  -p, --port=<port>           Kismet server port (default: 2501)
  -v, --verbose               Increase verbosity
  -f, --file=<file>           Write logs to file
  -n, --no-gps                Disable GPS support
  -l, --logging=<types>       Enable specific log types
      --daemonize             Run as background daemon
      --server-args=<args>    Extra server arguments
  
Sources are configured via kismet.conf or the
KISMET_SOURCES environment variable.

Available log types: pcap, alert, gps, dotview, netxml,
nettxt, wiglecsv, log

Example:
  kismet
  kismet -c /etc/kismet/kismet.conf
  kismet -l pcap,alert -f /tmp/kismet.log
