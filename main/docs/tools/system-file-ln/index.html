Parameters

Name Type Required Default Description Aliases
backup string No - make a backup of each existing destination file --backup
flag-b string No - like --backup but does not accept an argument -b
directory string No - allow the superuser to attempt to hard link directories, if supported by the system -d, -F, --directory
force string No - remove existing destination files -f, --force
interactive string No - prompt whether to remove destinations -i, --interactive
logical string No - dereference TARGETs that are symbolic links -L, --logical
no-dereference file No - treat LINK_NAME as a normal file if it is a symbolic link to a directory -n, --no-dereference
physical string No - make hard links directly to symbolic links -P, --physical
relative string No - with -s, create links relative to link location -r, --relative
symbolic string No - make symbolic links instead of hard links -s, --symbolic
suffix string No - override the usual backup suffix -S, --suffix
target-directory file No - specify the DIRECTORY in which to create the links -t, --target-directory
no-target-directory string No - treat LINK_NAME as a normal file always -T, --no-target-directory
verbose string No - print name of each linked file --help display this help and exit --version output version information and exit -v, --verbose
