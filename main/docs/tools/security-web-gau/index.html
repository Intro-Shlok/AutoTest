Examples

Fetch all URLs for a domain

gau example.com

Fetch URLs including subdomains

gau --subdomains example.com

Fetch from specific providers only

gau --providers wayback,otx example.com

Exclude common file extensions

gau --blacklist png,jpg,gif,css example.com

Multiple domains from stdin

cat domains.txt | gau --subdomains

Pipeline to probe live hosts

gau example.com | sort -u | httpx -status-code -title

Documentation

gau — Get All URLs

gau (Get All URLs) collects URLs from multiple public sources including the Wayback Machine, AlienVault OTX, and CommonCrawl. It provides comprehensive passive endpoint discovery for web security assessments and bug bounty hunting.

Common Pipelines

# Find parameters
gau target.com | grep "?.*=" | sort -u

# Find JavaScript files
gau target.com | grep "\.js$" | sort -u

# Pipeline probe live hosts
gau target.com | sort -u | httpx -status-code -title -json

# Full passive recon
echo target.com | gau --subdomains | sort -u | httpx -silent | nuclei -t cves/
