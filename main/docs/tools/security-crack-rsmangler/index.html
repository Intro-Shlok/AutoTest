Examples

Mangle a wordlist with default rules

rsmangler -i base.txt -o mangled.txt

Capitalise first letter and add numeric suffixes

rsmangler -i base.txt -o mangled.txt -c -n

Add prefix and suffix to every word

rsmangler -i base.txt -o mangled.txt -p "2024" -s "!"

Filter output by length and apply custom rules

rsmangler -i base.txt -l 6 -L 16 -m cap,leet,reverse -o output.txt

Uppercase all words with verbose logging

rsmangler -i base.txt -o output.txt -u -v

Documentation

rsmangler — Wordlist Mangling Tool

rsmangler takes a base wordlist and applies configurable transformation rules — capitalisation, uppercase, leet substitution, prefix/suffix, numeric append, reverse, and more — to produce a much larger set of password candidates for cracking workflows.
