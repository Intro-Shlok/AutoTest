Examples

Embed a file into a JPEG image

steghide embed -cf cover.jpg -ef secret.txt -p mypassphrase

Extract hidden data from a stego image

steghide extract -sf stego.jpg -p mypassphrase

Embed with encryption and compression

steghide embed -cf cover.wav -ef secret.txt -p mypass -e aes256 -z 9

Display info about a stego file

steghide info stego.jpg

Extract hidden data to a specific output file

steghide extract -sf stego.jpg -p mypass -xf extracted.txt

Embed using quiet mode

steghide embed -q -cf cover.bmp -ef data.zip -p mypass

Documentation

Steghide — Steganography Tool

Steghide embeds and extracts hidden data in JPEG, BMP, WAV, and AU files. It uses a passphrase for encryption and supports compression and various encryption algorithms.

Key Features

  • Multiple formats: JPEG, BMP, WAV, AU cover files
  • Encryption: AES, Twofish, and other algorithms via -e flag
  • Compression: Pre-embedding compression with adjustable level
  • Stego detection: The info subcommand reveals embedded data metadata
