Parameters

Name Type Required Default Description Aliases
all string No - include pseudo, duplicate, inaccessible file systems -a, --all
block-size integer No - scale sizes by SIZE before printing them; see SIZE format below; E.g., '-BM' prints sizes in units of 1,048,576 bytes -B, --block-size
human-readable string No - print sizes in powers of 1024 (e.g., 1023M) -h, --human-readable
si string No - print sizes in powers of 1000 (e.g., 1.1G) -H, --si
inodes string No - list inode information instead of block usage -i, --inodes
flag-k string No - like --block-size=1K -k
local string No - limit listing to local file systems --no-sync do not invoke sync before getting usage info (default) --output[=FIELD_LIST] use the output format defined by FIELD_LIST, or print all fields if FIELD_... -l, --local
portability string No - use the POSIX output format --sync invoke sync before getting usage info --total elide all entries insignificant to available space, and produce a grand total -P, --portability
type string No - limit listing to file systems of type TYPE -t, --type
print-type string No - print file system type -T, --print-type
exclude-type string No - limit listing to file systems not of type TYPE -x, --exclude-type
flag-v string No - (ignored) --help display this help and exit --version output version information and exit -v

Examples

Basic usage with all

df ${all}

Display help message

df --help

Print free disk space in a [h]uman-readable format.

df -h

Free disk space for [t]ype EXT2 file systems.

df -t ext2

Free disk space for filesystems, e[x]cluding EXT2.

df -x ext2

Show [i]node usage.

df -i

Show information about a distinct filesystem path.

df [PATH]

List [a]ll filesystems, + unreadable, duplicates, pseudo, and inaccessible.

df -a

Fetch a grand total of disk usage.

df --total
