Parameters

Name Type Required Default Description Aliases
dereference string No - follow links -L, --dereference
file-system string No - display file system status instead of file status --cached=MODE specify how to use cached attributes; useful on remote file systems. See MODE below -f, --file-system
format string No - use the specified FORMAT instead of the default; output a newline after each use of FORMAT --printf=FORMAT like --format, but interpret backslash escapes, and do not output a mandatory trailing new... -c, --format
terse string No - print the information in terse form --help display this help and exit --version output version information and exit -t, --terse
terse-2 string No - Set the terse-2 parameter --terse
terse-3 string No - Set the terse-3 parameter --terse, --file-system

Examples

Basic usage with dereference

stat ${dereference}

Display help message

stat --help

display numerical values for file permissions

stat -c '%a %n' *

Display only the octal permissions for the given directory. Great for tests.

stat --format='%a' /boot
