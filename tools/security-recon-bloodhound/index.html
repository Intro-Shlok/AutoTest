Examples

Full AD enumeration with all collection methods

bloodhound-python -d {{DOMAIN}} -u {{USER}} -p '{{PASSWORD}}' -ns {{IP}} -c all

Kerberos-based enumeration (no password needed on Linux)

bloodhound-python -d {{DOMAIN}} -u {{USER}} -k -no-pass -ns {{IP}} -c all

BloodHound.py remote ingestion from LDAP (no credentials)

bloodhound.py -c LDAP -d evilcorp.local -dc-ip 10.10.10.1 -ns 10.10.10.1

BloodHound.py with credentials via LDAP

bloodhound.py -u john -p password123 -d evilcorp.local -dc-ip 10.10.10.1 -ns 10.10.10.1 -c All

SharpHound.exe PowerShell execution on Windows

Invoke-BloodHound -CollectionMethod All -Domain evilcorp.local -ZipFile output.zip

Documentation

BloodHound — AD Attack Path Mapping

BloodHound is a single-page JavaScript web application that uses graph theory to reveal hidden relationships and attack paths in Active Directory environments. It ingests LDAP data and session information to build a comprehensive map of who-can-do-what.

Collection Methods

  • Group: Group membership
  • Session: Active user sessions
  • ComputerOnly: Computer details
  • LocalAdmin: Local admin groups
  • RDP: Remote Desktop users
  • DCOM: DCOM users
  • PSRemote: PowerShell Remote users
  • ACL: Access control entries
  • All: Everything above
