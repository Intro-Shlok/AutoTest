Examples

Basic usage with flag-4

sftp ${flag-4}

Display help message

sftp --help

Can be used to execute any command or file on a system, but without stdin/stdout/stderr.

sftp -D"touch /tmp/korewashere"

Proxy execution of specified command, can be used as a defensive evasion.

sftp -o ProxyCommand="{CMD}" .
