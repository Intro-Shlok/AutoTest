Examples

Generate all 6-8 character lowercase words

crunch 6 8 abcdefghijklmnopqrstuvwxyz -o wordlist.txt

Generate words with specific pattern

crunch 4 4 -t @@%%

Generate permutations of words

crunch 0 0 -p word1 word2 word3

Use a charset file

crunch 6 8 -f /usr/share/crunch/charset.lst mixalpha-numeric

Generate with start word

crunch 8 8 abcdefghijklmnopqrstuvwxyz -s aaaabbbb -o output.txt

Limit output to 50MB and compress

crunch 6 8 abcdefghijklmnopqrstuvwxyz -b 50MB -o START -z gzip

Documentation

Crunch — Wordlist Generator

Crunch is a wordlist generator that creates all possible combinations and permutations from specified character sets. It supports patterns with placeholders (@ for lowercase, , for uppercase, % for numbers, ^ for symbols), output compression (gzip, bzip2, lzma), file splitting, resume capability, and charset file integration.
