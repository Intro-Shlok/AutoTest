Examples

Attack all visible WiFi networks automatically

wifite -i wlan0

Attack only WPA networks

wifite -i wlan0 --wpa --no-wep

Attack specific BSSID

wifite -i wlan0 -b 00:11:22:33:44:55

Use only PMKID attack

wifite -i wlan0 --pmkid --no-wps --no-deauth

Skip deauth and WPS attacks

wifite -i wlan0 --no-deauth --no-wps

With custom wordlist

wifite -i wlan0 --wordlist /usr/share/wordlists/rockyou.txt

Attack 5GHz networks

wifite -i wlan0 -5 --no-wep

Use random MAC and cloaking

wifite -i wlan0 --random-mac --cloaking

Documentation

Wifite — Automated Wireless Auditor

Wifite automates WiFi auditing by orchestrating aircrack-ng, reaver, bully, and hcxdumptool. It targets WEP, WPA, and WPS-enabled networks with minimal user interaction.

Attack Modes

Mode Description
WEP ARP replay + packet injection, crack when enough IVs
WPA Deauth + handshake capture, crack with dictionary
WPS PIN brute-force via reaver/bully
PMKID PMKID capture via hcxdumptool
